# URBANopt Core

URBANopt core libraries and measures.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'urbanopt-core'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install 'urbanopt-core'
